# Web UI 更新说明（中文）- 第三轮增强

本轮继续往“盯盘工具”方向加强，而不是单纯堆统计。

## 本次新增功能

### 1. OPEN 持仓继续强化
- 增加 OPEN 持仓强化区域
- 展示：
  - 当前未平仓数
  - 最近交易日未平仓数
  - 最新交易日
- 未平仓表中补充：
  - 持有分钟数

### 2. 最近新增信号专区
- 新增“最近新增信号”面板
- 展示最新交易日最后 5 分钟窗口内的信号
- 便于快速发现刚出现的机会

### 3. 按代码聚合视图
- 新增按代码聚合表
- 展示：
  - 代码
  - 名称
  - BUY 次数
  - SELL 次数
  - 最新类型
  - 最新状态
  - 是否 OPEN
  - 最新时间

## 保留功能
- 自动刷新频率切换
- 最后刷新时间显示
- 新信号 / 新交易高亮
- 标题闪动提醒
- 信号/交易筛选

## 本轮意义
这一轮之后，页面更适合先回答三个问题：
1. 现在有没有持仓
2. 刚刚有没有新信号
3. 哪只基金最近最活跃
