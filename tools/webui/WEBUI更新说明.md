# Web UI 更新说明（中文）

本次更新主要补充了盯盘场景下更实用的交互功能，重点不是再堆统计，而是让页面更容易实时查看和筛选关键信号。

## 本次新增功能

### 1. 自动刷新增强
- 页面顶部显示：
  - 当前时间
  - 最后刷新时间
- 支持切换自动刷新频率：
  - 3 秒
  - 5 秒（默认）
  - 10 秒

### 2. 新信号 / 新交易高亮
- 刷新后新出现的信号行会高亮
- 刷新后新出现的交易行会高亮
- 方便在盯盘时快速定位新增变化

### 3. 浏览器标题闪动提醒
- 当刷新后发现有新的信号或新的交易时
- 浏览器标题会短暂闪动提醒
- 便于在标签页切走时知道页面有更新

### 4. 信号表筛选
支持：
- 按代码筛选（例如输入 513030）
- 按类型筛选：
  - 全部
  - BUY
  - SELL

### 5. 交易表筛选
支持：
- 按代码筛选
- 按交易类型筛选：
  - 全部
  - BUY
  - SELL
- 按状态筛选：
  - 全部状态
  - OPEN
  - CLOSED

## 本次没有改动的部分
- 没有改监控主逻辑
- 没有改买卖点策略
- 没有新增复杂前端依赖
- 仍然保持为轻量原生 HTML + JS 页面

## 备注
当前这版 Web UI 已经更接近“盯盘工具”而不是只读展示页，适合继续在此基础上迭代。
后续如果还要继续增强，优先建议：
1. 按基金代码聚合最近信号/交易
2. 单独的 OPEN 持仓看板
3. 最近 1 小时新增信号面板
4. 更明显的声音提醒 / 桌面提醒（如需要）
