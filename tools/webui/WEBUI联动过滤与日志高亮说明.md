# Web UI 联动过滤与日志高亮说明（中文）

## 本次新增功能

### 1. 代码联动过滤
现在支持点击代码进行联动过滤。

可点击位置：
- 按代码聚合视图中的代码行
- 最近新增信号卡片
- 最近 1 小时信号卡片

联动范围：
- 最近新增信号
- 最近 1 小时信号
- 最新信号表
- 最新交易表

页面顶部支持：
- 清除联动

### 2. 日志关键词高亮
日志面板会高亮：
- BUY
- SELL
- ERROR / Exception / Traceback
- 止盈 / 盈利保护 / profit
- 止损 / MA5 拐头止损 / loss

## 本轮意义
这轮之后，页面更适合：
- 针对某一只基金快速排查
- 从日志中快速定位交易动作和异常
- 从“多标的监控”快速切换到“单标的跟踪”
