# Web UI 更新说明（中文）- 第五轮增强

本轮重点继续偏实战：提高排查效率和盯盘效率。

## 本次新增功能

### 1. 日志关键词高亮
日志面板现在会高亮：
- BUY
- SELL
- ERROR / Exception / Traceback
- 止盈 / 盈利保护 / profit
- 止损 / MA5 拐头止损 / loss

作用：
- 不用再在长日志里肉眼找关键词
- 更容易快速定位交易动作和异常

### 2. 代码联动过滤
现在支持点击代码触发联动过滤：
- 点击“按代码聚合视图”里的某个代码
- 页面会联动过滤：
  - 最近新增信号
  - 最近 1 小时信号
  - 最新信号表
  - 最新交易表

并且支持：
- 点击“清除联动”恢复全量视图

### 3. 代码聚合视图更适合排查
代码聚合表继续保留表头排序，并且现在更适合配合联动过滤使用。

## 这轮适用场景
- 想快速看某只基金最近到底发生了什么
- 想快速从日志里找 BUY / SELL / ERROR
- 想在多标的同时监控时，更快收敛到单一代码排查
