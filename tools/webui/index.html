说明
1. 顶部“接口状态”用于区分核心接口是否正常，以及部分面板是否降级。
2. “最新类型”显示最近一条交易动作（BUY/SELL），“最新仓位状态”显示当前仓位是否 OPEN/CLOSED。
3. 默认刷新已调整为 10 秒；如果 Windows 上仍偶发不稳定，优先继续使用 10s 或 20s。
4. 日志区按时间从近到远显示，最新日志在最上方。
5. 日志区只显示最近 80 行，并且面板本身有滚动高度限制，避免无限输出。
这样就把“事件类型”和“仓位状态”拆成两层，避免把 BUY/SELL 和 OPEN/CLOSED 混成一个维度误读。
