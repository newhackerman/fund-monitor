# Web UI 更新说明（中文）- 第四轮增强

本轮继续围绕“盯盘可用性”增强，不做花哨功能，重点补三项。

## 本次新增功能

### 1. OPEN 持仓浮盈浮亏
- 在未平仓表中新增：
  - 最新价（基于最近分钟缓存最后价估算）
  - 浮盈亏百分比
- 说明：
  - 这里不是实时逐秒行情
  - 而是按最近分钟缓存最后价做近似估算
  - 适合作为盯盘参考

### 2. 最近 1 小时信号面板
- 新增“最近 1 小时信号面板”
- 用于快速查看最近一小时内出现过哪些信号
- 比只看最后 5 分钟更适合回看盘中节奏

### 3. 按代码聚合表支持排序
- 代码聚合表的表头现在可以点击排序
- 支持按以下字段排序：
  - 代码
  - 名称
  - BUY 次数
  - SELL 次数
  - 最新类型
  - 最新状态
  - 是否 OPEN
  - 最新时间

## 本轮意义
这一轮之后，Web UI 更接近一个能辅助盘中判断的监控页：
- 能看当前持仓的大概盈亏
- 能看最近 1 小时信号节奏
- 能按代码维度更方便筛查重点标的
