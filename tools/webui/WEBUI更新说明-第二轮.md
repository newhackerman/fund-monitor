# Web UI 更新说明（中文）- 第二轮增强

本轮重点不是增加花哨图表，而是把 Web UI 更往“可盯盘”方向推进。

## 本次新增功能

### 1. OPEN 持仓强化
- 单独增加 OPEN 持仓强化区域
- 展示：
  - 当前未平仓数
  - 最近交易日未平仓数
  - 最新交易日
- 未平仓表中新增：
  - 持有分钟数

### 2. 最近新增信号专区
- 增加“最近新增信号”区域
- 显示最新交易日最后 5 分钟窗口内的新增信号
- 便于快速查看最新变化，不需要去大表里找高亮

### 3. 按代码聚合视图
- 新增代码维度聚合表
- 展示：
  - 代码
  - 名称
  - BUY 次数
  - SELL 次数
  - 最新类型
  - 最新状态
  - 是否 OPEN
  - 最新时间

## 保留功能
- 自动刷新频率切换（3s / 5s / 10s）
- 最后刷新时间显示
- 新信号 / 新交易高亮
- 浏览器标题闪动提醒
- 信号表筛选
- 交易表筛选

## 相关文件
- `skills/fund-monitor/tools/webui/index.html`
- `skills/fund-monitor/tools/webui_server.py`

## 本轮价值
这一轮之后，Web UI 更适合：
- 先看当前有没有 OPEN 持仓
- 再看最近有没有新信号
- 再按代码判断某只基金是否频繁触发
